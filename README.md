**CITASalud** es una plataforma web de **autoagendamiento de citas médicas** que permite a los usuarios gestionar sus citas de manera rápida y sencilla.

**Funcionalidades principales**
- Agendar citas médicas.
- Modificar citas médicas.
- Cancelar citas, indicando un motivo.
- Visualizar estadísticas de asistencia e inasistencia.
